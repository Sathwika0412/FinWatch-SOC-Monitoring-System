# FinWatch SOC Monitoring System

A Python-based Security Operations Center (SOC) simulation project that analyzes authentication logs and detects potential cybersecurity threats.

## Features
- Brute-force attack detection
- After-hours login anomaly detection
- Phishing activity simulation
- Incident severity classification
- Automated incident report generation

## Technologies Used
- Python
- Regex
- Log Analysis
- Basic Threat Detection Logic

## Project Goal
To simulate real-world SIEM functionality and demonstrate SOC analyst skills in log monitoring and incident response.
